# 传习录

# 徐爱录

  • 至善是心之本体，只是明明德至精至一处便是
    • ‘天理’即是‘明德’， ‘穷理’即是‘明明德’
  • 心即理也，此心无私欲之蔽，即是天理
  • 身之主宰便是心，心之所发便是意，意之本体便是知，意之所在便是物

# 陆澄录

  • 日间工夫觉纷扰，则静坐；觉懒看书，则且看书：是亦因病而药
  • 是徒知静养，而不用克己功夫也。如此，临事便要倾倒。人须在事上磨，方立得住，方能静亦定，动亦定
  • 知者行之始，行者知之成，圣学只一个功夫，知、行不可分作两事
  • 虚灵不昧，众理具而万事出
  • 此念如树之根芽，立志者，长立此善念而已
  • 克己须要扫除廓清，一毫不存，方是；有一毫在，则众恶相引而来

# 薛侃录

  • 无善无恶者理之静，有善有恶者气之动。不动于气，即无善无恶，是谓至善
  • 悔悟是去病之药；然以改之为贵，若留滞于中，则又因药发病

# 答顾东桥书

  • 知之真切笃行处即是行，行之明觉精察处即是知
    • 笃者，敦实笃厚之意。已行矣，而敦笃其行，不息其功之谓尔

# 答陆原静书

  • 虽则圣贤别有真乐，而亦常人之所同有，但常人有之而不自知，反自求许多忧苦，自加迷弃

# 黄修易录

  • 上智绝少，学者无超入圣人之理，一起一伏，一进一退，自是工夫节次

# 黄省曾

  • 汝若以厌外物之心去求之静，是反养成一个骄惰之气了；汝若不厌外物，复于静处涵养，却好

# 钱德洪录

  • 天命之谓性，率性之谓道，修道之谓教
  • 你未看此花时，此花与汝心同归于寂；你来看此花时，则此花颜色一时明白起来：便知此花不在你的心外
  • 不可以云能蔽日，教天不要生云。
    • 七情顺其自然之流行，皆是良知之用，不可分辨善恶，但不可有着。
    • 七情有着，俱谓之欲，俱为良知之蔽。
  • 无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物

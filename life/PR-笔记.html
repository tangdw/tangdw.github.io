# pr 快捷键

  • crtl + alt + n 新建项目

  • ctrl + o 打开

  • ctrl + i 导入

  • ctrl + m 导出

  • ctrl + q 退出

  • alt + 滚轮 缩放

  • ctrl + k 在时间线上切分素材

  • ctrl + d 默认视频转场

  • ctrl + t 新建字幕

  • ctrl + l/c/r 对齐

  • ctrl + ` 切换全屏
